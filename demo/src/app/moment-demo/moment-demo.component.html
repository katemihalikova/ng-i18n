This module includes formatting pipes for any input supported by moment.js. Each pipe provides a different output based on its name and format, but can also accept arbitrary formatting tokens defined in moment.js documentation.

Pipes Demo Table

Table lists all supported pipes. The last column shows the output of the particular pipe applied to a datetime ISO string. You can change format to see different outputs. To change value of the ISO string, see below the table.

Pipe Format Result
izDate {{model | izMDate: izMDateFormat}}
izTime {{model | izMTime: izMTimeFormat}}
izDateTime {{model | izMDateTime: izMDateTimeFormat}}
izYearMonth {{model | izMYearMonth: izMYearMonthFormat}}
izWeekday {{model | izMWeekday: izMWeekdayFormat}}
