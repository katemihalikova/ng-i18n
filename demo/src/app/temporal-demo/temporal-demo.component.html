This module includes formatting pipes for Temporal API. Each pipe accepts different Temporal objects based on their intended usage.

Pipes Demo Table

Table lists all supported pipes and their possible input Temporal type. The last column shows the output of the particular pipe applied to a Temporal object. You can change format to see different outputs. To change value of used Temporal objects, see below the table.

Pipe Temporal Object Format Result
izDate Temporal.PlainDate {{plainDate | izDate: izDateFormat}}
Temporal.PlainDateTime {{plainDateTime | izDate: izDateFormat}}
izZonedDate Temporal.ZonedDateTime {{zonedDateTime | izZonedDate: izDateFormat}}
izLocalDate Temporal.ZonedDateTime {{zonedDateTime | izLocalDate: izDateFormat}}
Temporal.Instant {{instant | izLocalDate: izDateFormat}}
izTime Temporal.PlainTime {{plainTime | izTime: izTimeFormat}}
Temporal.PlainDateTime {{plainDateTime | izTime: izTimeFormat}}
izZonedTime Temporal.ZonedDateTime {{zonedDateTime | izZonedTime: izTimeZonedFormat}}
izLocalTime Temporal.ZonedDateTime {{zonedDateTime | izLocalTime: izTimeZonedFormat}}
Temporal.Instant {{instant | izLocalTime: izTimeZonedFormat}}
izDateTime Temporal.PlainDateTime {{plainDateTime | izDateTime: izDateTimeFormat}}
izZonedDateTime Temporal.ZonedDateTime {{zonedDateTime | izZonedDateTime: izDateTimeZonedFormat}}
